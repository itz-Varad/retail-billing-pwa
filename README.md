# Retail Billing PWA

An initial-stage Progressive Web App (PWA) for retail billing and invoice generation.

## Features
- Retail bill generation
- Customer details input
- Itemized billing
- Mobile-friendly interface
- PWA support

## Tech Stack
- Next.js / React
- TypeScript
- Progressive Web App (PWA)
- Vercel Deployment
